  251  <h1 class="entry-title" itemprop="headline">How to Install TeamViewer on Ubuntu</h1><div class="entry-meta">
  252  <center style="padding-bottom:20px;padding-top:40px;">
  253  <div align="center" id="tecmint_leaderboard_article_top" data-google-query-id="COjSgJHSj-8CFV7XEQgddtUG5w">
  254  <script data-cfasync="false" type="text/javascript">
  255  freestar.config.enabled_slots.push({ placementName: "tecmint_leaderboard_article_top", slotId: "tecmint_leaderboard_article_top" });
  256  </script>
  257  <div id="google_ads_iframe_/15184186/tecmint_leaderboard_article_top_0__container__" style="border: 0pt none; width: 728px; height: 90px;"></div></div>
  258  </center>
  259  <div class="entry-content" itemprop="text">
  260  <p><strong>TeamViewer</strong> is a cross-platform, proprietary application that allows a user to remotely gain access to another user’s desktop, share the desktop and even allow <a title="10 SCP Commands to Transfer Files/Folders in Linux" href="https://www.tecmint.com/scp-commands-examples/" target="_blank" rel="noopener noreferrer">file transfer between computers</a> over an internet connection. It’s a popular application among helpdesk support staff and comes in handy when helping out remote users who are stuck and cannot find worthwhile assistance.</p>
  261  <p>In this guide, you will learn how to install <strong>TeamViewer</strong> on <strong>Ubuntu 20.04</strong> and <strong>Ubuntu 18.04</strong> <strong>LTS</strong> versions.</p>
  262  <h3>Installing TeamViewer in Ubuntu</h3>
  263  <p>Before getting started, it’s recommended to update your system packages. This will ensure that you start off on a clean slate. So open your terminal and issue the command below.</p>
  264  <pre>$ sudo apt update -y  &amp;&amp; sudo apt upgrade -y
  265  </pre>
  266  <p>Once you are through with updating your system, head over to the official <a href="https://www.teamviewer.com/en-us/download/linux/" target="_blank" rel="noopener nofollow noreferrer">Teamviewer’s download page</a> and download Teamviewer’s Debian package. Alternatively, you can copy the download link and download the Debian package using the following <a title="10 Wget (Linux File Downloader) Command Examples in Linux" href="https://www.tecmint.com/10-wget-command-examples-in-linux/" target="_blank" rel="noopener noreferrer">wget command</a> as shown.</p>
  267  <pre>$ sudo wget https://download.teamviewer.com/download/linux/teamviewer_amd64.deb
  268  </pre>
  269  <p>Upon successfully downloading the file, you can verify its existence using the <a title="15 Basic ‘ls’ Command Examples in Linux" href="https://www.tecmint.com/15-basic-ls-command-examples-in-linux/" target="_blank" rel="noopener noreferrer">ls command</a> as shown.</p>
  270  <pre>$ ls | grep teamviewer
  271  <strong>teamviewer_amd64.deb</strong>
  272  </pre>
  273  <center style="padding-bottom: 45px;padding-top:35px;"><div align="center" id="tecmint_incontent" data-google-query-id="CPvIk5HSj-8CFV7XEQgddtUG5w" style="flex-direction: column;"><div id="google_ads_iframe_/15184186/tecmint_incontent_0__container__" style="border: 0pt none;"><iframe id="google_ads_iframe_/15184186/tecmint_incontent_0" title="3rd party ad content" name="google_ads_iframe_/15184186/tecmint_incontent_0" width="728" height="90" scrolling="no" marginwidth="0" marginheight="0" frameborder="0" sandbox="allow-forms allow-popups allow-popups-to-escape-sandbox allow-same-origin allow-scripts allow-top-navigation-by-user-activation" srcdoc="" style="border: 0px; vertical-align: bottom;" data-google-container-id="4" data-load-complete="true"></iframe></div><div class="__fs-ancillary" style="--childWidth:728px;"><div class="__fs-branding"><a href="https://freestar.com/?utm_medium=ad_container&amp;utm_source=branding&amp;utm_name=tecmint_incontent" target="_blank"><img src="https://a.pub.network/core/imgs/fslogo-green.svg" alt="Freestar"></a></div></div></div></center><p>To install <strong>TeamViewer</strong>, run the command shown. This will install TeamViewer alongside other dependencies.</p>
  274  <pre>$ sudo apt install ./teamviewer_amd64.deb
  275  </pre>
  276  <figure id="attachment_37139" aria-describedby="caption-attachment-37139" style="width: 726px" class="wp-caption aligncenter"><a href="https://www.tecmint.com/wp-content/uploads/2020/04/Install-TeamViewer-on-Ubuntu.png"><img src="https://www.tecmint.com/wp-content/plugins/lazy-load/images/1x1.trans.gif" data-lazy-src="https://www.tecmint.com/wp-content/uploads/2020/04/Install-TeamViewer-on-Ubuntu.png" loading="lazy" class="size-full wp-image-37139" alt="Install TeamViewer on Ubuntu" width="736" height="472"><noscript><img loading="lazy" class="size-full wp-image-37139" src="https://www.tecmint.com/wp-content/uploads/2020/04/Install-TeamViewer-on-Ubuntu.png" alt="Install TeamViewer on Ubuntu" width="736" height="472" /></noscript></a><figcaption id="caption-attachment-37139" class="wp-caption-text">Install TeamViewer on Ubuntu</figcaption></figure>
  277  <p>When prompted to continue with the installation, type <code>‘Y’</code> for <strong>Yes</strong> and hit the ‘<strong>ENTER</strong>’ button.</p>
  278  <p>Once the installation is complete, you can proceed to Launch <strong>Teamviewer</strong>. To launch <strong>Teamviewer</strong>, run the command below on the terminal.</p>
  279  <pre>$ teamviewer
  280  </pre>
  281  <p>Also, you can use the application manager to search and launch the <strong>TeamViewer</strong> application as shown.</p>
  282  <figure id="attachment_37140" aria-describedby="caption-attachment-37140" style="width: 505px" class="wp-caption aligncenter"><a href="https://www.tecmint.com/wp-content/uploads/2020/04/Search-TeamViewer-on-Ubuntu.png"><img src="https://www.tecmint.com/wp-content/plugins/lazy-load/images/1x1.trans.gif" data-lazy-src="https://www.tecmint.com/wp-content/uploads/2020/04/Search-TeamViewer-on-Ubuntu.png" loading="lazy" class="size-full wp-image-37140" alt="Launch TeamViewer from Application Manager" width="515" height="266"><noscript><img loading="lazy" class="size-full wp-image-37140" src="https://www.tecmint.com/wp-content/uploads/2020/04/Search-TeamViewer-on-Ubuntu.png" alt="Launch TeamViewer from Application Manager" width="515" height="266" /></noscript></a><figcaption id="caption-attachment-37140" class="wp-caption-text">Launch TeamViewer from Application Manager</figcaption></figure>
  283  <p>Once launched, Accept the EULA agreement as shown.</p>
  284  <figure id="attachment_37141" aria-describedby="caption-attachment-37141" style="width: 568px" class="wp-caption aligncenter"><a href="https://www.tecmint.com/wp-content/uploads/2020/04/TeamViewer-License-Agreement.png"><img src="https://www.tecmint.com/wp-content/plugins/lazy-load/images/1x1.trans.gif" data-lazy-src="https://www.tecmint.com/wp-content/uploads/2020/04/TeamViewer-License-Agreement.png" loading="lazy" class="size-full wp-image-37141" alt="TeamViewer License Agreement" width="578" height="355"><noscript><img loading="lazy" class="size-full wp-image-37141" src="https://www.tecmint.com/wp-content/uploads/2020/04/TeamViewer-License-Agreement.png" alt="TeamViewer License Agreement" width="578" height="355" /></noscript></a><figcaption id="caption-attachment-37141" class="wp-caption-text">TeamViewer License Agreement</figcaption></figure>
  285  <p>And finally, you’ll get TeamViewer’s user interface and shown below.</p>
  286  <figure id="attachment_37142" aria-describedby="caption-attachment-37142" style="width: 878px" class="wp-caption aligncenter"><a href="https://www.tecmint.com/wp-content/uploads/2020/04/Running-TeamViewer-on-Ubuntu.png"><img src="https://www.tecmint.com/wp-content/plugins/lazy-load/images/1x1.trans.gif" data-lazy-src="https://www.tecmint.com/wp-content/uploads/2020/04/Running-TeamViewer-on-Ubuntu.png" loading="lazy" class="size-full wp-image-37142" alt="Running TeamViewer on Ubuntu" width="888" height="523" srcset="https://www.tecmint.com/wp-content/uploads/2020/04/Running-TeamViewer-on-Ubuntu.png 888w, https://www.tecmint.com/wp-content/uploads/2020/04/Running-TeamViewer-on-Ubuntu-768x452.png 768w" sizes="(max-width: 888px) 100vw, 888px"><noscript><img loading="lazy" class="size-full wp-image-37142" src="https://www.tecmint.com/wp-content/uploads/2020/04/Running-TeamViewer-on-Ubuntu.png" alt="Running TeamViewer on Ubuntu" width="888" height="523" srcset="https://www.tecmint.com/wp-content/uploads/2020/04/Running-TeamViewer-on-Ubuntu.png 888w, https://www.tecmint.com/wp-content/uploads/2020/04/Running-TeamViewer-on-Ubuntu-768x452.png 768w" sizes="(max-width: 888px) 100vw, 888px" /></noscript></a><figcaption id="caption-attachment-37142" class="wp-caption-text">Running TeamViewer on Ubuntu</figcaption></figure>
  287  <p>To make a remote connection to another user, simply provide them with your <strong>Teamviewer ID</strong> and <strong>password</strong>. The user will insert the <strong>ID</strong> at the ‘<strong>Insert partner ID</strong>’ text field after which they will click on the ‘<strong>Connect</strong>’ button. Later, they will be prompted for the <strong>password</strong> which will then give them a remote connection to your desktop.</p>
  288  <p>And that’s how you install <strong>TeamViewer</strong> on <strong>Ubuntu</strong>. Thanks for taking the time on this article.</p>

  505  <h1 class="entry-title" itemprop="headline">How to Install TeamViewer on Ubuntu</h1><div class="entry-meta">
  506  <center style="padding-bottom:20px;padding-top:40px;">
  507  <div align="center" id="tecmint_leaderboard_article_top" data-google-query-id="COjSgJHSj-8CFV7XEQgddtUG5w">
  508  <script data-cfasync="false" type="text/javascript">
  509  freestar.config.enabled_slots.push({ placementName: "tecmint_leaderboard_article_top", slotId: "tecmint_leaderboard_article_top" });
  510  </script>
  511  <div id="google_ads_iframe_/15184186/tecmint_leaderboard_article_top_0__container__" style="border: 0pt none; width: 728px; height: 90px;"></div></div>
  512  </center>
  513  <div class="entry-content" itemprop="text">
  514  <p><strong>TeamViewer</strong> is a cross-platform, proprietary application that allows a user to remotely gain access to another user’s desktop, share the desktop and even allow <a title="10 SCP Commands to Transfer Files/Folders in Linux" href="https://www.tecmint.com/scp-commands-examples/" target="_blank" rel="noopener noreferrer">file transfer between computers</a> over an internet connection. It’s a popular application among helpdesk support staff and comes in handy when helping out remote users who are stuck and cannot find worthwhile assistance.</p>
  515  <p>In this guide, you will learn how to install <strong>TeamViewer</strong> on <strong>Ubuntu 20.04</strong> and <strong>Ubuntu 18.04</strong> <strong>LTS</strong> versions.</p>
  516  <h3>Installing TeamViewer in Ubuntu</h3>
  517  <p>Before getting started, it’s recommended to update your system packages. This will ensure that you start off on a clean slate. So open your terminal and issue the command below.</p>
  518  <pre>$ sudo apt update -y  &amp;&amp; sudo apt upgrade -y
  519  </pre>
  520  <p>Once you are through with updating your system, head over to the official <a href="https://www.teamviewer.com/en-us/download/linux/" target="_blank" rel="noopener nofollow noreferrer">Teamviewer’s download page</a> and download Teamviewer’s Debian package. Alternatively, you can copy the download link and download the Debian package using the following <a title="10 Wget (Linux File Downloader) Command Examples in Linux" href="https://www.tecmint.com/10-wget-command-examples-in-linux/" target="_blank" rel="noopener noreferrer">wget command</a> as shown.</p>
  521  <pre>$ sudo wget https://download.teamviewer.com/download/linux/teamviewer_amd64.deb
  522  </pre>
  523  <p>Upon successfully downloading the file, you can verify its existence using the <a title="15 Basic ‘ls’ Command Examples in Linux" href="https://www.tecmint.com/15-basic-ls-command-examples-in-linux/" target="_blank" rel="noopener noreferrer">ls command</a> as shown.</p>
  524  <pre>$ ls | grep teamviewer
  525  <strong>teamviewer_amd64.deb</strong>
  526  </pre>
  527  <center style="padding-bottom: 45px;padding-top:35px;"><div align="center" id="tecmint_incontent" data-google-query-id="CPvIk5HSj-8CFV7XEQgddtUG5w" style="flex-direction: column;"><div id="google_ads_iframe_/15184186/tecmint_incontent_0__container__" style="border: 0pt none;"><iframe id="google_ads_iframe_/15184186/tecmint_incontent_0" title="3rd party ad content" name="google_ads_iframe_/15184186/tecmint_incontent_0" width="728" height="90" scrolling="no" marginwidth="0" marginheight="0" frameborder="0" sandbox="allow-forms allow-popups allow-popups-to-escape-sandbox allow-same-origin allow-scripts allow-top-navigation-by-user-activation" srcdoc="" style="border: 0px; vertical-align: bottom;" data-google-container-id="4" data-load-complete="true"></iframe></div><div class="__fs-ancillary" style="--childWidth:728px;"><div class="__fs-branding"><a href="https://freestar.com/?utm_medium=ad_container&amp;utm_source=branding&amp;utm_name=tecmint_incontent" target="_blank"><img src="https://a.pub.network/core/imgs/fslogo-green.svg" alt="Freestar"></a></div></div></div></center><p>To install <strong>TeamViewer</strong>, run the command shown. This will install TeamViewer alongside other dependencies.</p>
  528  <pre>$ sudo apt install ./teamviewer_amd64.deb
  529  </pre>
  530  <figure id="attachment_37139" aria-describedby="caption-attachment-37139" style="width: 726px" class="wp-caption aligncenter"><a href="https://www.tecmint.com/wp-content/uploads/2020/04/Install-TeamViewer-on-Ubuntu.png"><img src="https://www.tecmint.com/wp-content/plugins/lazy-load/images/1x1.trans.gif" data-lazy-src="https://www.tecmint.com/wp-content/uploads/2020/04/Install-TeamViewer-on-Ubuntu.png" loading="lazy" class="size-full wp-image-37139" alt="Install TeamViewer on Ubuntu" width="736" height="472"><noscript><img loading="lazy" class="size-full wp-image-37139" src="https://www.tecmint.com/wp-content/uploads/2020/04/Install-TeamViewer-on-Ubuntu.png" alt="Install TeamViewer on Ubuntu" width="736" height="472" /></noscript></a><figcaption id="caption-attachment-37139" class="wp-caption-text">Install TeamViewer on Ubuntu</figcaption></figure>
  531  <p>When prompted to continue with the installation, type <code>‘Y’</code> for <strong>Yes</strong> and hit the ‘<strong>ENTER</strong>’ button.</p>
  532  <p>Once the installation is complete, you can proceed to Launch <strong>Teamviewer</strong>. To launch <strong>Teamviewer</strong>, run the command below on the terminal.</p>
  533  <pre>$ teamviewer
  534  </pre>
  535  <p>Also, you can use the application manager to search and launch the <strong>TeamViewer</strong> application as shown.</p>
  536  <figure id="attachment_37140" aria-describedby="caption-attachment-37140" style="width: 505px" class="wp-caption aligncenter"><a href="https://www.tecmint.com/wp-content/uploads/2020/04/Search-TeamViewer-on-Ubuntu.png"><img src="https://www.tecmint.com/wp-content/plugins/lazy-load/images/1x1.trans.gif" data-lazy-src="https://www.tecmint.com/wp-content/uploads/2020/04/Search-TeamViewer-on-Ubuntu.png" loading="lazy" class="size-full wp-image-37140" alt="Launch TeamViewer from Application Manager" width="515" height="266"><noscript><img loading="lazy" class="size-full wp-image-37140" src="https://www.tecmint.com/wp-content/uploads/2020/04/Search-TeamViewer-on-Ubuntu.png" alt="Launch TeamViewer from Application Manager" width="515" height="266" /></noscript></a><figcaption id="caption-attachment-37140" class="wp-caption-text">Launch TeamViewer from Application Manager</figcaption></figure>
  537  <p>Once launched, Accept the EULA agreement as shown.</p>
  538  <figure id="attachment_37141" aria-describedby="caption-attachment-37141" style="width: 568px" class="wp-caption aligncenter"><a href="https://www.tecmint.com/wp-content/uploads/2020/04/TeamViewer-License-Agreement.png"><img src="https://www.tecmint.com/wp-content/plugins/lazy-load/images/1x1.trans.gif" data-lazy-src="https://www.tecmint.com/wp-content/uploads/2020/04/TeamViewer-License-Agreement.png" loading="lazy" class="size-full wp-image-37141" alt="TeamViewer License Agreement" width="578" height="355"><noscript><img loading="lazy" class="size-full wp-image-37141" src="https://www.tecmint.com/wp-content/uploads/2020/04/TeamViewer-License-Agreement.png" alt="TeamViewer License Agreement" width="578" height="355" /></noscript></a><figcaption id="caption-attachment-37141" class="wp-caption-text">TeamViewer License Agreement</figcaption></figure>
  539  <p>And finally, you’ll get TeamViewer’s user interface and shown below.</p>
  540  <figure id="attachment_37142" aria-describedby="caption-attachment-37142" style="width: 878px" class="wp-caption aligncenter"><a href="https://www.tecmint.com/wp-content/uploads/2020/04/Running-TeamViewer-on-Ubuntu.png"><img src="https://www.tecmint.com/wp-content/plugins/lazy-load/images/1x1.trans.gif" data-lazy-src="https://www.tecmint.com/wp-content/uploads/2020/04/Running-TeamViewer-on-Ubuntu.png" loading="lazy" class="size-full wp-image-37142" alt="Running TeamViewer on Ubuntu" width="888" height="523" srcset="https://www.tecmint.com/wp-content/uploads/2020/04/Running-TeamViewer-on-Ubuntu.png 888w, https://www.tecmint.com/wp-content/uploads/2020/04/Running-TeamViewer-on-Ubuntu-768x452.png 768w" sizes="(max-width: 888px) 100vw, 888px"><noscript><img loading="lazy" class="size-full wp-image-37142" src="https://www.tecmint.com/wp-content/uploads/2020/04/Running-TeamViewer-on-Ubuntu.png" alt="Running TeamViewer on Ubuntu" width="888" height="523" srcset="https://www.tecmint.com/wp-content/uploads/2020/04/Running-TeamViewer-on-Ubuntu.png 888w, https://www.tecmint.com/wp-content/uploads/2020/04/Running-TeamViewer-on-Ubuntu-768x452.png 768w" sizes="(max-width: 888px) 100vw, 888px" /></noscript></a><figcaption id="caption-attachment-37142" class="wp-caption-text">Running TeamViewer on Ubuntu</figcaption></figure>
  541  <p>To make a remote connection to another user, simply provide them with your <strong>Teamviewer ID</strong> and <strong>password</strong>. The user will insert the <strong>ID</strong> at the ‘<strong>Insert partner ID</strong>’ text field after which they will click on the ‘<strong>Connect</strong>’ button. Later, they will be prompted for the <strong>password</strong> which will then give them a remote connection to your desktop.</p>
  542  <p>And that’s how you install <strong>TeamViewer</strong> on <strong>Ubuntu</strong>. Thanks for taking the time on this article.</p>
